# Tennis Match Analytics (Quarto)

This folder contains a Quarto project that renders the EDA and ML analysis into a single website.

## Quick start

1. Ensure Quarto is installed and available in PATH.
2. From this directory, run:

```bash
quarto render
```

This will create the `_site` folder with the HTML output.

## Project pages

- `index.qmd`: overview and final narrative summary.
- `hm_datasciende.ipynb`: full analysis notebook rendered by Quarto.
